INTEL FUSION TOOL
==================

Intelligence Processing System for Military Operations
Converts unstructured chat messages → Structured database

INCLUDED FILES:
---------------
1. intel_fusion.html         - Main application (open in browser)
2. test_messages.txt          - 30 sample messages for testing
3. bookmarklet_setup.txt      - Browser automation guide
4. QUICK_START.txt            - Getting started in 1 minute
5. README.txt                 - This file

INSTALLATION:
-------------
None required. Just open intel_fusion.html in your browser.

QUICK START:
------------
1. Open intel_fusion.html
2. Copy test messages from test_messages.txt
3. Paste into text box
4. Click "Process & Add"

WHAT IT DOES:
-------------
✓ Extracts coordinates (MGRS format)
✓ Identifies equipment types (tanks, APCs, artillery)
✓ Classifies activities (movement, engagement, position)
✓ Detects priority levels
✓ Prevents duplicates
✓ Exports to Excel

WORKS WITH:
-----------
✓ NATO APP-11 SALUTE reports
✓ Casual chat messages  
✓ Mixed/partial formats
✓ Real-time operations
✓ Air-gapped networks
