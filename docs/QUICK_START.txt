INTEL FUSION - QUICK START GUIDE
=================================

WHAT IT IS:
Automatically converts chat messages into structured intelligence database.

WHAT YOU GET:
✅ intel_fusion.html - The tool (open in browser)
✅ test_messages.txt - 30 sample messages to test
✅ bookmarklet_setup.txt - Browser automation setup
✅ This guide

================================
1-MINUTE SETUP
================================

1. Open "intel_fusion.html" in your browser (Chrome/Edge)
2. Copy messages from "test_messages.txt"
3. Paste into the text box
4. Click "Process & Add"
5. Done! View your intelligence database

================================
BASIC USAGE
================================

METHOD 1: COPY-PASTE (Easiest)
1. Copy chat messages
2. Paste into Intel Fusion
3. Click "Process & Add"
4. View extracted intelligence in table

METHOD 2: BOOKMARKLET (Auto-extract)
1. Follow setup in "bookmarklet_setup.txt"
2. Enable "Auto Mode" in Intel Fusion
3. Click bookmarklet on chat page
4. Messages auto-extract

================================
WHAT IT EXTRACTS
================================

FROM THIS:
"UAV2 spotted 3 tanks moving north, grid 35V MD 85243 89003"

TO THIS:
┌─────────────┬──────────────────┬──────────┬──────────┐
│ DTG         │ Coordinate       │ Activity │ Equipment│
├─────────────┼──────────────────┼──────────┼──────────┤
│ 191505DJAN26│ 35V MD 85243 ... │ MOVEMENT │ TANK     │
└─────────────┴──────────────────┴──────────┴──────────┘

Extracts:
✓ Coordinates (MGRS format)
✓ Date-time groups
✓ Equipment types
✓ Activity (movement, engagement, position)
✓ Unit strength
✓ Priority level

================================
TIPS & TRICKS
================================

GETTING BETTER RESULTS:
- Include coordinates when possible
- Mention equipment types
- Add time information
- Use SALUTE format when you can

AVOIDING DUPLICATES:
- Tool auto-detects duplicates
- Based on content similarity
- Safe to paste same chat multiple times
- Only new messages added

MOBILE USE:
- Works on phones/tablets
- Table hides some columns (tap row for full details)
- Copy-paste works best
- Bookmarklet may not work

================================
TROUBLESHOOTING
================================

NO DATA EXTRACTED?
→ Messages might be too short (min 20 characters)
→ Check for coordinates or intel keywords
→ Try longer, more detailed messages

TOO MANY DUPLICATES?
→ This is normal and good (prevents re-processing)
→ Tool is working correctly

EXPORT NOT WORKING?
→ Check browser allows downloads
→ File saves as .tsv (open with Excel)
→ Try different browser if issues

BOOKMARKLET FAILS?
→ Use copy-paste instead
→ Some chat pages aren't compatible
→ Manual method always works
