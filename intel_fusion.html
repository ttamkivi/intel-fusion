ℹ️ What is Intel Fusion? (Quick Guide)

Intel Fusion converts raw battlefield chat messages into a structured, searchable intelligence database. It runs entirely in your browser — no internet, no installation, no data leaves your machine.

New in v2.1:

Auto Mode — Pulls/scrapes messages from chat tabs at configurable frequency (PULL model)
JSON Database — Full-featured save/load with duplicate detection and metadata
NATO Reports — INTSUM, SPOTREP, and SITREP in military standard format
Excel XLS Export — Real Excel 2003 XML with bold headers, red HIGH priority rows, and column widths
TSV & Clipboard — Quick export to SharePoint, Google Sheets, or local analysis

How it works:

1. Click 📡 Start Auto-Scan — scrapes chat tabs automatically at set intervals
2. Each chat channel runs in a separate browser tab — all are scanned
3. Filter, search, and view details with equipment photos
4. Generate NATO reports (INTSUM, SPOTREP, BDA, etc.) or export to Excel/TSV
5. Manual paste available as fallback when auto-scan isn't possible

Security: Zero network calls (Content Security Policy enforced). No cookies, no storage, no tracking. Safe for air-gapped and classified networks.
